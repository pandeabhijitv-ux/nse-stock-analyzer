# Asset Placeholders

This folder should contain the following assets for the app:

## Required Assets

1. **icon.png** (1024x1024)
   - App icon for both iOS and Android
   - Should be a square image

2. **splash.png** (1284x2778 or similar)
   - Splash screen shown when app launches
   - Can be a branded screen or logo

3. **adaptive-icon.png** (1024x1024)
   - Android adaptive icon
   - Should have important content in center circle

4. **favicon.png** (48x48)
   - Web favicon if using Expo web

## Creating Assets

For now, you can:

1. Use Expo's default assets (already configured)
2. Create custom assets using design tools like:
   - Figma
   - Adobe Illustrator
   - Canva
   - Online icon generators

3. Replace the placeholder paths in `app.json` with your custom assets

## Online Tools

- **Icon Generator**: https://appicon.co/
- **Splash Screen**: https://www.figma.com/templates/
- **Free Icons**: https://www.flaticon.com/

The app will work with Expo's default assets until you add custom ones.
